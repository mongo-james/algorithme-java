# Algorithmes numériques en Java


##  Description

Ce projet regroupe plusieurs algorithmes numériques écrits en Java pour la **résolution de systèmes d'équations linéaires** et l'**interpolation**.

Développé dans le cadre de travaux en méthodes numériques, ce code permet de comprendre et d'implémenter des algorithmes fondamentaux.
Et de faciliter le calcul pour les utilisateurs.
---

##  Algorithmes inclus

### Résolution de systèmes linéaires
| Algorithme | Description |
|------------|-------------|
| **PGauss.java** | Méthode d'élimination de Gauss |
| **PGaussJordan.java** | Méthode de Gauss-Jordan |
| **PSeidel.java** | Méthode itérative de Gauss-Seidel |
| **PJacobi.java** | Méthode itérative de Jacobi |

### Factorisation
| Algorithme | Description |
|------------|-------------|
| **Cholesky.java** | Factorisation de Cholesky pour matrices symétriques définies positives |

### Interpolation
| Algorithme | Description |
|------------|-------------|
| **Lagrange.java** | Interpolation polynomiale de Lagrange |

- Java JDK 8 ou supérieur

```bash
